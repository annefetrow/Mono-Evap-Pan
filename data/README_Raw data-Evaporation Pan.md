
### Air Temp

Stores the temperature of air above the evaporation pan. The file data type should be `.csv` and it can be converted manually from `.xlsx`.

| Data Structure |                      |                          |
|---------------------------|----------------------|--------------------------|
| #                         | Date Time, GMT-07:00 | EV pan Level, Millimeter |
| 1                         | MM/DD/YY HH:MM       | num                      |
| 2                         | MM/DD/YY HH:MM       |
